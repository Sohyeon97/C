# C언어 7주차

- C언어 7주차 실습

https://www.acmicpc.net/problem/10952 << 문제

```
#include <stdio.h>

int main() {
    int a, b;
    scanf("%d %d", &a, &b);
    printf("%d\n", a + b);
    return 0;
}
```

ㄴ 문제 답 : 위 코드에서 scanf() 함수를 통해 두 정수 A와 B를 입력받고, printf() 함수를 이용하여 A+B의 결과를 출력합니다.<br> 
             코드의 마지막으로 return 0;을 통해 프로그램이 정상적으로 종료되었음을 알립니다.
